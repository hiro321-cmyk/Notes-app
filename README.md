#How it works

in this note-app user can simply click the +add note button and it will show the interface to write the note and after that user can simply click to save and it will save into the local storage of the browser. and if user want to delete the note simply the user can click on "x" button. and even when the user log out of the website and do refersh the user can still get the acces of their save note.

#features
The app can do the following mention things
-Add new note
-save note in local storage
-Delete note
